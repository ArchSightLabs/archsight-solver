# 温度作用后续评估

本文用于 v1.5.0 发布目标的温度作用边界收口。结论是：v1.5.0 不把 ArchSight Solver 扩展成热分析平台，只继续支持已经落地的平面框架构件均匀温差，并把后续扩展作为独立评估项管理。

## 当前支持

- 结构类型：二维平面框架。
- 荷载形式：构件均匀温差。
- 分析假定：线弹性静力，小变形，构件截面温度均匀。
- 输出口径：节点位移、支座反力、杆端内力和结果图。

## v1.5.0 不承诺

- 截面温度梯度。
- 瞬态热传导。
- 热-结构耦合迭代。
- 混凝土徐变、收缩或温度松弛。
- 桥梁专用温度场和规范组合设计。
- 理想桁架杆件截面温度梯度或弯曲温度效应。

## 后续评估矩阵

| 方向 | 模型相容性 | 工程价值 | v1.5.0 结论 |
|---|---:|---:|---|
| 平面桁架杆件均匀温差 | 中 | 中 | 可作为后续增强项，但需先补二力杆轴向温差 benchmark |
| 平面框架截面温度梯度 | 中 | 高 | 需要新增截面温度分布、热曲率换算、符号约定和公开算例 |
| 梁系截面温度梯度 | 中 | 高 | 需要独立定义输入契约、图形表达和教材解析基准 |
| 瞬态热传导 / 热-结构耦合 | 低 | 高 | 不进入当前开源核心路线 |

## 验收要求

任何温度扩展进入实现前，必须同时补齐：

- ASMS-JSON / API 字段契约。
- 至少一个可复核 benchmark 或教材解析算例。
- DOCX / XLSX 计算书表达。
- 前端工程图图层和结果来源说明。
- 明确的非目标说明，防止把结构静力求解器包装成热分析平台。
