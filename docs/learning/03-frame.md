# 平面框架分析入门

平面框架由梁、柱、斜撑等构件在同一平面内连接形成。与桁架不同，框架构件不仅能承受轴力，还能承受剪力和弯矩。

平面框架的典型节点自由度是：

```text
ux / uy / rz
```

也就是水平位移、竖向位移和平面内转角。转角自由度使框架构件能够传递弯矩。

## 适合场景

框架适合回答：

- 门式刚架在水平荷载下顶部侧移多大。
- 梁柱节点处弯矩如何分布。
- 柱脚固定、铰接或转动弹簧如何影响位移和弯矩。
- 构件分布荷载如何转换为杆端内力。
- 哪根构件弯矩最大。

典型对象包括门式刚架、单层平面框架、多跨平面框架、带斜撑框架和框架梁退化验证模型。

不适合直接用当前平面框架解决：

- 三维空间框架。
- 楼板、墙、壳等面单元问题。
- 非线性、稳定、动力、施工阶段或规范设计问题。
- 理想铰接桁架的纯轴力问题。

## 基本假定

- 线弹性。
- 小变形。
- 静力分析。
- 构件位于二维平面内。
- 节点包含 `ux / uy / rz` 自由度。
- 构件可恢复杆端轴力、剪力和弯矩。

## 核心对象

### 节点

节点是构件连接、荷载作用和支座约束的位置。框架节点不仅可以平移，还可以转动。

### 构件

构件连接两个节点，可以代表梁、柱或斜撑。构件通常包含弹性模量 `E`、截面面积 `A`、截面惯性矩 `I`、材料和截面编号。

框架构件同时依赖轴向刚度 `EA` 和弯曲刚度 `EI`。

### 支座

框架支座可以约束 `ux`、`uy` 和 `rz`。

| 支座 | 约束直觉 | 常见反力 |
|---|---|---|
| 铰支座 | 约束水平、竖向位移，允许转动 | 水平反力、竖向反力 |
| 滚支座 | 约束一个平动方向 | 一个方向反力 |
| 固定支座 | 约束水平、竖向和转角 | 水平反力、竖向反力、弯矩反力 |
| 转动弹簧 | 部分限制转角 | 与转角相关的弯矩反力 |

### 荷载

框架荷载包括节点水平力、节点竖向力、节点力矩、构件均布荷载和构件集中荷载。

框架必须特别注意荷载坐标系。全局 X/Y 荷载与构件局部 x/y 荷载不同，斜构件上的局部 y 向荷载不等于全局竖向荷载。

## 关键结果

### 节点位移

节点位移包括 `ux`、`uy` 和 `rz`。门式刚架或单层框架常关注顶部水平位移。

“层间位移角”只应在明确存在楼层概念时使用，不能把任意框架的节点水平位移都叫层间位移角。

### 支座反力

框架支座反力可能包括水平反力、竖向反力和支座弯矩反力。固定柱脚常出现弯矩反力；铰支座通常不提供弯矩反力。

### 杆端内力

框架构件结果常以杆端内力表达：

| 内力 | 含义 | 单位 |
|---|---|---|
| 轴力 | 沿构件轴线拉压 | kN |
| 剪力 | 垂直构件轴线 | kN |
| 弯矩 | 构件端部弯曲内力矩 | kN·m |

## 结果图

| 图形 | 重点检查 |
|---|---|
| 结构图 | 节点、构件、支座和几何是否正确 |
| 荷载图 | 节点荷载、构件荷载和局部方向是否正确 |
| 变形图 | 整体侧移、梁挠曲和支座约束是否合理 |
| 弯矩图 | 柱脚、梁端、梁柱节点和控制构件是否合理 |
| 剪力图 | 构件横向内力是否与荷载匹配 |
| 轴力图 | 柱、梁和斜撑拉压是否符合传力路径 |

## 与梁系和桁架的区别

| 项目 | 梁系 | 平面桁架 | 平面框架 |
|---|---|---|---|
| 建模心智 | 跨段、支座、荷载 | 节点、杆件、支座、荷载 | 节点、构件、支座、荷载 |
| 自由度 | `v / θz` | `ux / uy` | `ux / uy / rz` |
| 构件内力 | 剪力、弯矩 | 轴力 | 轴力、剪力、弯矩 |
| 主图 | 弯矩、剪力、挠度 | 轴力、节点位移 | 弯矩、剪力、轴力、变形 |

## 常见误区

- 把框架当作桁架，只看轴力，忽略弯矩。
- 把桁架当作框架，给所有杆件输出弯矩图。
- 忽略节点转角自由度，导致刚接和铰接概念混乱。
- 把局部坐标荷载误认为全局竖向荷载。
- 对普通框架随意使用“层间位移角”术语。
- 柱脚支座类型录错，导致侧移和弯矩完全不同。

## 与未来三维框架的关系

二维平面框架是未来三维空间框架的基础，但不是简单增加一个 Z 坐标。

二维框架自由度为：

```text
ux / uy / rz
```

三维框架自由度为：

```text
ux / uy / uz / rx / ry / rz
```

三维还会引入局部 y/z 方向、强轴/弱轴、扭转刚度、空间梁单元、三维支座和三维结果图。
