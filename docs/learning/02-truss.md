# 平面桁架分析入门

平面桁架是由若干直杆通过节点连接形成的二维杆系结构。理想桁架的核心假定是：

- 杆件两端近似铰接。
- 荷载主要作用在节点上。
- 每根杆件主要只承受轴力。
- 不把杆件弯矩和剪力作为主结果。

## 适合场景

桁架适合回答：

- 屋架或桥式桁架中每根杆件是受拉还是受压。
- 哪根杆件轴力最大。
- 哪个节点位移最大。
- 支座反力是否与外荷载平衡。
- 杆件轴应力是否偏高。

典型对象包括屋架、桥式桁架、Pratt 桁架、Warren 桁架、Howe 桁架和悬挑桁架。

不适合直接用桁架解决：

- 梁柱刚接、构件需要传递弯矩的问题。
- 分布荷载直接作用在杆件中部且弯曲明显的问题。
- 节点连接刚度显著的问题。
- 三维空间桁架。

## 基本假定

平面桁架节点只有：

```text
ux / uy
```

也就是水平位移和竖向位移。桁架节点没有转角自由度，支座也不提供转动约束。

杆件按两力杆处理。两力杆只在两端受力，杆内力沿杆件轴线。因此桁架的主结果是轴力、轴应力、节点位移和支座反力。

## 核心对象

### 节点

节点是杆件连接、荷载作用和支座约束的位置。节点通常包含坐标、编号、节点荷载和支座信息。

### 杆件

杆件连接两个节点，通常包含弹性模量 `E`、截面面积 `A`、材料或截面编号。桁架杆件主要刚度是轴向刚度 `EA / L`。

### 支座

桁架支座只能约束平动自由度，例如 `ux`、`uy` 或两者。因为节点没有转角自由度，所以桁架支座不应出现转动约束。

### 荷载

桁架经典荷载是节点荷载。如果存在屋面或杆件自重，通常应先等效为节点荷载，并在模型或计算书中说明等效口径。

## 关键结果

### 节点位移

节点位移表示每个节点在 X、Y 方向移动多少。异常大的节点位移常提示支座不足、机构或输入错误。

### 支座反力

支座反力用于检查整体平衡。水平荷载应有对应水平反力；竖向荷载应有对应竖向反力。

### 杆件轴力

杆件轴力是桁架最核心的结果。项目口径中通常拉为正、压为负。

### 杆件轴应力

轴应力为轴力除以截面面积：

```text
σ = N / A
```

轴应力可帮助发现高风险杆件，但不能替代规范中的稳定、连接和构造设计。

## 结果图

| 图形 | 重点检查 |
|---|---|
| 结构图 | 节点、杆件编号和连接拓扑是否正确 |
| 荷载图 | 荷载是否作用在节点，方向和大小是否正确 |
| 轴力图 | 拉压状态、最大轴力杆件和零力杆是否合理 |
| 节点位移图 | 变形方向、最大位移节点和支座约束是否合理 |

## 零力杆

零力杆是在当前工况下轴力为零或接近零的杆件。它不一定无用，可能用于其他荷载工况、保持几何稳定或满足构造要求。

公开结果中更准确的说法是“当前工况下轴力接近零”，而不是“无效杆件”。

## 常见误区

- 要求桁架输出弯矩图或剪力图作为主结果。
- 把分布荷载直接加在杆件中部，却仍按理想桁架解释。
- 只看轴力绝对值，不看拉压符号。
- 把线弹性轴力结果当成完整设计结论，忽略受压稳定和连接设计。
- 支座约束不足形成机构，却误认为是正常大变形。
