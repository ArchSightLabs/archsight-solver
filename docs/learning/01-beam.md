# 梁系分析入门

梁系分析关注一根或多跨梁在荷载作用下的弯曲、剪力、弯矩、挠度和支座反力。

在 ArchSight Solver 中，梁系的建模心智是：

```text
跨段 + 支座 + 荷载 + 材料截面
```

内部计算可以映射为节点和梁单元，但用户入口应优先保持连续梁、简支梁和悬臂梁的专业表达。

## 适合场景

梁系适合回答：

- 简支梁在均布荷载下最大挠度是多少。
- 悬臂梁端部集中力会产生多大端部挠度和固定端弯矩。
- 连续梁中间支座附近是否出现负弯矩。
- 最大剪力和最大弯矩出现在什么位置。
- 支座反力是否与总荷载平衡。

不适合直接用梁系解决：

- 梁柱刚接和明显水平侧移问题，应使用平面框架。
- 三角形铰接杆系的纯轴力问题，应使用平面桁架。
- 三维、动力、稳定、非线性和规范设计问题。

## 基本假定

- 线弹性。
- 小变形。
- 静力分析。
- 主要关注竖向挠度、剪力、弯矩和支座反力。
- 常见自由度为 `v / θz`，即竖向位移和截面转角。

## 核心对象

### 跨段

跨段是相邻支座或控制点之间的一段梁。跨段通常包含长度 `L`、弹性模量 `E`、截面惯性矩 `I` 等参数。

### 支座

梁系支座限制梁的竖向位移或转角。

| 支座 | 约束直觉 |
|---|---|
| 简支 / 铰支 | 限制竖向位移，允许转动 |
| 滚支 | 提供竖向支承，允许滑动和转动 |
| 固定 / 固结 | 同时限制位移和转角 |
| 中间支座 | 连续梁跨间支承，影响负弯矩 |

### 荷载

梁系常见荷载包括均布荷载、集中荷载、线性分布荷载和叠加荷载。荷载方向和单位必须明确，常见单位为 kN 或 kN/m。

## 关键结果

### 支座反力

支座反力是支座为保持梁平衡而提供的力。对称简支梁在对称荷载下，左右反力通常相等；非对称荷载或连续梁中，反力分配会改变。

### 剪力

剪力是梁截面上的横向内力，单位通常为 kN。剪力图用于检查支座反力、集中荷载跳变和分布荷载区间的变化趋势。

### 弯矩

弯矩是让梁弯曲的内力矩，单位通常为 kN·m。简支梁跨中常出现正弯矩，连续梁中间支座附近常出现负弯矩。

### 挠度

挠度是梁弯曲后的位移，常用 mm 表示。它是服务性指标，反映梁是否变形过大。

## 结果图

| 图形 | 重点检查 |
|---|---|
| 结构图 | 跨长、支座位置、支座类型是否正确 |
| 荷载图 | 荷载位置、方向、大小和单位是否正确 |
| 剪力图 | 支座反力和集中荷载处是否有合理跳变 |
| 弯矩图 | 最大弯矩、负弯矩区间和控制截面是否合理 |
| 挠度曲线 | 最大挠度位置、方向和支座处约束是否合理 |

## 读图顺序

1. 先看结构图和荷载图，确认模型没有录错。
2. 再看支座反力，检查整体平衡。
3. 看剪力图，确认荷载引起的跳变和斜率变化。
4. 看弯矩图，定位最大弯矩和负弯矩。
5. 看挠度曲线，检查最大挠度和变形方向。

## 常见误区

- 把挠度当成力。挠度是位移，不是荷载。
- 只看最大弯矩，不看挠度。强度和服务性是不同问题。
- 认为支座反力一定平均分配。只有对称结构和对称荷载下才可能平均。
- 把连续梁中间支座负弯矩误认为错误。
- 把普通梁系入口复杂化为节点-杆件对象树。
