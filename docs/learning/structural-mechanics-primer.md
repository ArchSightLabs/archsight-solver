# 三类分析对象总览与结构力学术语地图

本文帮助读者理解 ArchSight Solver 当前公开核心中的三类分析对象：梁系、平面桁架和平面框架。

结构力学求解器回答的核心问题是：

```text
几何模型 + 材料截面 + 支座约束 + 荷载
        ↓
自由度与刚度
        ↓
位移 / 转角
        ↓
内力 / 应力 / 支座反力
        ↓
结构图 / 荷载图 / 结果图 / 计算书
```

## 三类对象的核心区别

| 分析对象 | 适合解决的问题 | 典型自由度 | 主结果指标 | 不应作为主指标 |
|---|---|---|---|---|
| 梁系 | 一根或多跨梁在竖向荷载下如何弯曲 | `v / θz` | 挠度、弯矩、剪力、支座反力 | 杆件轴应力 |
| 平面桁架 | 铰接杆系中每根杆是受拉还是受压 | `ux / uy` | 节点位移、杆件轴力、杆件轴应力、支座反力 | 弯矩、剪力 |
| 平面框架 | 梁柱刚接体系如何侧移、弯曲和传力 | `ux / uy / rz` | 节点位移、杆端轴力/剪力/弯矩、支座反力 | 只看轴力 |

一句话记忆：

- 梁系：看梁怎么弯。
- 桁架：看杆件是拉还是压。
- 框架：看梁柱整体怎么侧移、弯曲和传力。

## 基础对象

### 节点

节点是构件连接、支座约束或荷载作用的位置。

- 梁系用户通常按跨段、支座、荷载建模，内部再映射为梁节点。
- 桁架节点是杆件铰接点，通常只含 `ux / uy` 平动自由度。
- 框架节点可以平移和转动，通常含 `ux / uy / rz`。

### 跨段、杆件和构件

| 术语 | 使用对象 | 含义 |
|---|---|---|
| 跨段 | 梁系 | 相邻支座或控制点之间的一段梁 |
| 杆件 | 平面桁架 | 两端铰接、主要承受轴力的直杆 |
| 构件 | 平面框架 | 梁、柱、斜撑等，可承受轴力、剪力和弯矩 |

### 荷载

荷载是作用在结构上的外力或外力矩。常见类型包括集中力、均布荷载、线性分布荷载、节点荷载和构件荷载。

桁架的经典假定要求荷载主要作用在节点上。梁系和框架可以处理作用在梁或构件上的分布荷载。

### 支座

支座限制结构某些方向的运动，并因此产生支座反力。

| 支座类型 | 直觉 | 可能反力 |
|---|---|---|
| 滚动支座 | 限制一个方向位移，允许滑动和转动 | 一个方向反力 |
| 铰支座 | 限制两个平动方向，允许转动 | 水平、竖向反力 |
| 固定支座 | 限制平动和转动 | 水平反力、竖向反力、弯矩反力 |
| 弹性支座 | 像弹簧一样抵抗运动 | 与位移或转角相关的反力 |

完整支座类型、自由度约束和图例说明见：[支座类型与图例](supports.md)。

## 基础结果

### 位移、转角和挠度

位移是点的位置变化；转角是截面或节点的转动；挠度通常指梁或构件弯曲方向上的位移。

挠度不是力，而是变形量，常用 mm 表示。挠度过大可能导致开裂、积水、门窗变形、设备无法正常工作或观感问题。

### 支座反力

支座反力是支座为了阻止结构运动而“推回去”的力或力矩。它用于检查整体平衡，也为基础、连接件和支承构造提供设计输入。

### 内力

内力是构件内部为了抵抗外部荷载而产生的力。二维杆系中最常见的内力有：

| 内力 | 单位 | 直觉 |
|---|---|---|
| 轴力 | kN | 沿构件轴线拉或压 |
| 剪力 | kN | 垂直构件轴线，使截面有错动趋势 |
| 弯矩 | kN·m | 让构件弯曲的力矩 |

轴力让杆件变长或变短；剪力让相邻截面有错动趋势；弯矩让构件弯起来。

### 应力和刚度

应力是单位面积上的内力。桁架中常见轴应力：

```text
轴应力 σ = 轴力 N / 截面面积 A
```

刚度表示抵抗变形的能力。常见参数：

| 参数 | 含义 | 主要影响 |
|---|---|---|
| `E` | 弹性模量 | 材料本身硬不硬 |
| `A` | 截面面积 | 轴向拉压刚度 `EA` |
| `I` | 截面惯性矩 | 弯曲刚度 `EI` |

桁架主要依赖 `EA`；梁和框架的弯曲行为强依赖 `EI`。

## 常见图形

| 图形 | 作用 | 常见对象 |
|---|---|---|
| 结构图 | 检查几何、编号、支座和连接关系 | 三类对象 |
| 荷载图 | 检查荷载位置、方向、大小和单位 | 三类对象 |
| 挠度图 / 变形图 | 检查结构受力后的变形形态 | 三类对象 |
| 剪力图 | 展示梁或框架构件的横向内力 | 梁系、框架 |
| 弯矩图 | 展示梁或框架构件的弯曲内力 | 梁系、框架 |
| 轴力图 | 展示杆件或构件的拉压状态 | 桁架、框架 |

## 项目中的专业底线

- 梁系优先关注：挠度、弯矩、剪力、支座反力。
- 平面桁架优先关注：节点位移、杆件轴力、杆件轴应力、支座反力。
- 平面框架优先关注：水平位移、竖向位移、杆端轴力/剪力/弯矩、支座反力。
- 桁架不得把弯矩、剪力作为主结果指标。
- 梁系入口应保持跨段、支座、荷载心智，不应强迫用户按节点-杆件对象树建模。
- 框架和桁架虽然都可用节点-构件模型表达，但自由度、支座、荷载和结果图完全不同。

## 延伸阅读

荷载方向、挠度方向、局部坐标和内力正负号容易在梁系、框架和桁架之间混淆。具体约定见 [符号与方向约定](sign-conventions.md)。
