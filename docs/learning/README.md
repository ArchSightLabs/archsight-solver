# 结构力学入门

本目录面向三类读者：

- 非土木专业背景的开发者和产品参与者。
- 正在使用 ArchSight Solver 学习梁系、平面桁架和平面框架的学生。
- 想为项目贡献模板、benchmark、图表或导出字段的开源贡献者。

这些文档只说明当前开源核心中的线弹性静力分析概念，不替代结构设计规范、课程教材或具备资质人员的工程复核。

## 选择分析对象

| 任务 | 优先选择 |
|---|---|
| 简支梁、悬臂梁、连续梁的挠度、弯矩、剪力和支座反力 | 梁系 |
| 屋架、桥式桁架、铰接杆系的节点位移、杆件轴力和轴应力 | 平面桁架 |
| 门式刚架、梁柱刚接体系的节点位移、杆端轴力/剪力/弯矩 | 平面框架 |

如果一个问题需要三维空间构件、动力、稳定、非线性或规范设计结论，它超出了当前公开核心范围。
