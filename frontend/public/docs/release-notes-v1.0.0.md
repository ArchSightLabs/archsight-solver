# ArchSight 结构力学求解器更新说明

版本号：v1.0.0
版本类型：首个公开版本  
适用对象：结构工程师、土木/结构方向教师、高校学生、结构工程工具开发者

## 版本定位

v1.0.0 是 ArchSight 结构力学求解器的第一个完整版本。本版本以 Web 工作台形式交付梁系、平面桁架、平面框架三类典型结构分析能力，重点解决“快速建模、即时计算、结果复核、计算书导出”的闭环问题。

本版本不是规范设计软件，也不替代注册结构工程师的审查判断。它面向结构力学计算、教学演示、方案阶段复核和工程工具原型验证。

## 一、工作台基础能力

### 三类分析对象

- 梁系：支持连续梁、简支梁、悬臂梁等常见梁系模型。
- 平面桁架：支持典型屋架、悬臂桁架、自定义节点-杆件模型。
- 平面框架：支持门式刚架、显式节点-构件建模、节点荷载、构件荷载与杆端释放等建模场景。

### 三阶段主工作区

- 参数建模：录入结构体系、材料、几何、荷载、支座、截面等输入参数。
- 结构计算：运行求解并查看结果摘要、受力变形、位移与内力结果。
- 敏感性分析：对关键参数施加扰动，观察响应指标变化。

### 专业结果总览

- 结构计算页默认进入“全部结果”。
- 梁系总览一次展示：结果摘要、受力变形、挠度曲线、弯矩图、剪力图。
- 平面桁架总览一次展示：结果摘要、受力变形、节点水平位移、节点竖向位移、杆件轴力。
- 平面框架总览一次展示：结果摘要、受力变形、节点水平位移、节点竖向位移、构件轴力/剪力/弯矩/位移曲线。
- 原单项页签保留，便于对某个指标进行局部放大复核。

## 二、梁系分析能力

### 建模输入

- 支持跨段长度、弹性模量、截面惯性矩、荷载参数等核心输入。
- 支持常用材料：Q235、Q345、C30、C35、C40、C50，以及自定义材料。
- 支持节点与支座由系统自动派生，降低基础梁系建模成本。
- 支持典型案例导入，便于快速进入标准工况。

### 荷载与计算

- 支持均布荷载、集中荷载、线性分布荷载等常见梁系荷载形式。
- 使用梁单元法与 Euler-Bernoulli 梁理论进行线弹性静力求解。
- 执行 GPa 到 Pa、kN/m 到 N/m、cm^4 到 m^4 等单位换算。

### 结果输出

- 输出允许挠度、最大挠度、最大挠度位置、校核状态。
- 输出受力变形、支座反力、挠度曲线、弯矩图、剪力图。
- 支持 Word 与 Excel 计算书导出。

## 三、平面桁架分析能力

### 建模输入

- 支持典型屋架与自定义桁架模型。
- 支持节点坐标、杆件连接、截面面积、材料与支座约束。
- 支持节点荷载与等效节点荷载预处理。

### 计算与结果

- 输出节点水平位移、节点竖向位移、合位移。
- 输出支座反力。
- 输出杆件轴力、轴应力与受拉/受压状态。
- 输出最大节点位移、最大轴力与控制节点/杆件。

### 专业约束

- 桁架模块以节点位移、杆件轴力、杆件轴应力、支座反力为主指标。
- 不把弯矩作为桁架主结果，避免偏离桁架计算假定。

## 四、平面框架分析能力

### 建模输入

- 支持门式刚架快捷参数建模。
- 支持显式节点、构件、支座、荷载的自定义建模。
- 支持构件截面、材料编号、节点约束、构件荷载与节点荷载。
- 支持荷载工况与荷载组合结果展示。

### 计算与结果

- 输出节点水平位移、节点竖向位移、转角。
- 输出支座反力。
- 输出构件杆端轴力、剪力、弯矩。
- 输出构件测站曲线，包括轴力、剪力、弯矩与位移。
- 支持最大位移、最大竖向位移、最大弯矩与控制节点/构件信息。

### 扩展建模支持

- 支持显式二维框架模型。
- 支持部分杆端释放、弹性支座或转动弹簧等高级建模字段的后端契约。
- 支持统一 API 响应包络，兼容后续前后端演进。

## 五、敏感性分析能力

- 支持对梁系、平面桁架、平面框架执行参数扰动分析。
- 支持响应指标选择，例如最大挠度、最大弯矩、最大剪力、最大节点位移、最大杆件轴力等。
- 支持多步扰动计算，返回扰动曲线与响应变化。
- 支持在计算书中纳入敏感性分析结果。

## 六、模板库与工作流

- 支持保存当前工况为项目模板。
- 支持恢复模板、复制模板、删除模板。
- 支持设置基准模板，便于快速回到标准算例。
- 内置梁系、平面桁架、平面框架典型案例。
- 支持工作台参数与分析对象联动，切换结构体系时自动清理旧结果，降低误读风险。

## 七、计算书与数据导出

### Word 计算书

- 支持导出梁系、平面桁架、平面框架计算书。
- 包含输入参数、结果摘要、结构图、曲线图、表格结果与敏感性分析结果。
- 使用专业结构工程术语表达计算结果。

### Excel 表格

- 支持导出关键结果表格。
- 梁系导出摘要、参数、荷载、结果明细等内容。
- 桁架导出节点结果、杆件结果、约定说明等内容。
- 框架导出节点结果、构件结果、构件曲线数据等内容。

## 八、接口与工程化

- 提供 `/api/calculate` 计算接口。
- 提供 `/api/sensitivity` 敏感性分析接口。
- 提供 `/api/export` Word / Excel 导出接口。
- 统一返回 `version: v1` 的 API 包络，包含 request、model、results、diagnostics、errors、meta 等字段。
- 前端使用 Vite + React + TypeScript + Tailwind CSS v4 + ECharts。
- 后端使用 Python + Flask + NumPy + Pandas + Openpyxl。

## 九、验证与质量门禁

- 后端覆盖梁系、平面桁架、平面框架、错误契约、基准算例、导出计算书等测试。
- 前端覆盖模板库、求解 payload、API 包络、框架构件曲线、文本建模、受力变形工具函数等单元测试。
- 已配置 GitHub Actions，包含后端测试、前端 lint、前端单元测试与前端构建。

## 十、已知限制

- 当前聚焦线弹性静力分析，不提供规范配筋、构件承载力设计或施工安全验算。
- 当前不包含影响线、动力分析、稳定分析、三维杆系求解和 OpenSeesPy 等外部引擎接入。
- 当前不是多用户协作系统，不包含权限、审签、项目归档等企业平台能力。
- 计算结果应由具备结构工程背景的使用者复核后再用于工程决策。

## 十一、推荐使用场景

- 教学中的连续梁、平面桁架、平面框架课堂演示。
- 方案阶段快速核对位移、弯矩、剪力、轴力与支座反力。
- 结构工程工具的原型验证与开源计算核扩展。
- 对典型结构算例进行基准化回归测试。
