版本 v1.0.0

ArchSight 结构力学求解器更新说明

版本类型：首个公开版本
适用对象：结构工程师、土木/结构方向教师、高校学生、结构工程工具开发者
v1.0.0 是 ArchSight 结构力学求解器的第一个完整版本。本版本以 Web 工作台形式交付梁系、平面桁架、平面框架三类典型结构分析能力，重点解决“快速建模、即时计算、结果复核、计算书导出”的闭环问题。

一、工作台基础能力

三类分析对象

  • 梁系：连续梁、简支梁、悬臂梁。
  • 平面桁架：典型屋架、悬臂桁架、自定义桁架。
  • 平面框架：门式刚架、显式节点-构件建模。

三阶段主工作区

  • 参数建模：录入材料、几何、支座、荷载、截面。
  • 结构计算：运行求解并查看结果。
  • 敏感性分析：观察参数扰动对响应指标的影响。

专业结果总览

  • 结构计算页默认进入“全部结果”。
  • 一次查看摘要、结构预览、位移与内力结果。
  • 保留单项页签，便于局部复核。

二、梁系分析能力

  • 支持连续梁、简支梁、悬臂梁等常见梁系模型。
  • 支持跨段长度、弹性模量、截面惯性矩、均布荷载、集中荷载、线性分布荷载等输入。
  • 执行 GPa 到 Pa、kN/m 到 N/m、cm^4 到 m^4 等单位换算。
  • 输出允许挠度、最大挠度、最大挠度位置、支座反力、挠度曲线、弯矩图、剪力图。

三、平面桁架分析能力

  • 支持典型屋架、自定义桁架、节点坐标、杆件连接、截面面积、材料与支座约束。
  • 输出节点水平位移、节点竖向位移、合位移、支座反力、杆件轴力、轴应力。
  • 桁架模块以节点位移、杆件轴力、杆件轴应力、支座反力为主指标，不把弯矩作为桁架主结果。

四、平面框架分析能力

  • 支持门式刚架快捷建模和显式节点-构件建模。
  • 支持节点荷载、构件荷载、支座约束、荷载工况与荷载组合。
  • 输出节点水平位移、节点竖向位移、转角、支座反力、杆端轴力、剪力、弯矩。
  • 输出构件测站曲线，包括轴力、剪力、弯矩与位移。

五、敏感性分析能力

  • 支持对梁系、平面桁架、平面框架执行参数扰动分析。
  • 支持最大挠度、最大弯矩、最大剪力、最大节点位移、最大杆件轴力等响应指标。
  • 支持多步扰动计算，返回扰动曲线与响应变化，并可纳入计算书。

六、模板库与导出

  • 支持保存、恢复、复制、删除项目模板，并设置基准模板。
  • 内置梁系、平面桁架、平面框架典型案例。
  • 支持 Word 与 Excel 计算书导出，包含输入参数、结果摘要、结构图、曲线图、表格结果与敏感性分析结果。

七、接口与工程化

  • 提供 /api/calculate 计算接口。
  • 提供 /api/sensitivity 敏感性分析接口。
  • 提供 /api/export Word / Excel 导出接口。
  • 统一返回 version: v1 的 API 包络，包含 request、model、results、diagnostics、errors、meta 等字段。

八、已知限制

  • 当前聚焦线弹性静力分析，不提供规范配筋、构件承载力设计或施工安全验算。
  • 当前不包含影响线、动力分析、稳定分析、三维杆系求解和外部引擎接入。
  • 计算结果应由具备结构工程背景的使用者复核后再用于工程决策。
